# IPNI Taxonomy API Documentation

This project provides a **FastAPI-based RESTful API** for querying taxonomic data from **IPNI**. The data is stored in a **MySQL database**, accessible via **phpMyAdmin**, and is containerized using **Docker**.

## Key Features
- **FastAPI** for API access
- **SQL ORM (SQLAlchemy)** for database management
- **MySQL Server** with **phpMyAdmin** integration
- **Docker** for deployment and containerization
- **CRUD operations** for taxonomic data management
